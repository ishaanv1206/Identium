# @identium/dashboard

A premium, glassmorphic web interface for visualizing and managing Identium persistent agents.

## Features

- **Quick Start Guide**: Instant instructions for MCP integration.
- **Agent Passport**: High-fidelity visualization of GAIDs, traits, and Personality DNA.
- **Interactive DNA**: Procedural SVG visualization of agent personality vectors.
- **Ownership Management**: UI for claiming and monitoring agent residents.

## Development

```bash
npm run dev
```

## Build
